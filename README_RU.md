# 🔥 Исследование файрвола Ирана — Что работает и почему

> Реальное исследование на основе многочасового тестирования во время активного интернет-блэкаута в Иране (март 2026).
> Никакой теории. Никаких догадок. Только то, что реально работало — и почему.

---

## 🧠 Главный вывод (кратко)

**Протокол важен меньше, чем РАСПОЛОЖЕНИЕ сервера.**

После 40+ неудачных конфигов на европейских серверах (VLESS+Reality, VMess, Shadowsocks и т.д.) мы выяснили, что **все рабочие конфиги вели на серверы, физически расположенные внутри Ирана**. Иранский DPI не обязан анализировать трафик — во время блэкаута он просто дропает или замедляет соединения с зарубежными IP на уровне инфраструктуры.

---

## 🗺️ Как работает иранский файрвол (упрощённо)

```
Твой телефон
    │
    ▼
[Иранский DPI / Файрвол]
    │
    ├─── Зарубежный IP? → Блокировка / Таймаут ❌
    │
    └─── Иранский IP? → Пропускает ✅
                            │
                            ▼
                    [Иранский сервер]
                            │
                            ▼
                    [Интернет / Заграница]
```

Во время активного блэкаута файрволу не нужно анализировать протокол или шифрование — он просто убивает соединения с зарубежными IP.

## 🔑 Паттерн который работает

Все рабочие конфиги имеют общие черты:

1. **Сервер физически в Иране** — тегеранские датацентры (AbrArvan, Soroush Rasanheh, FanAvaran Mihan Mizban)
2. **Нестандартный порт** — 9901, 2013, 17858, 58203 (не 80/443 которые активно мониторятся)
3. **Без TLS** — как ни странно, TLS fingerprint привлекает внимание DPI больше чем обычный трафик
4. **Иранские домены в Host/SNI** — `myket.ir`, `aparat.com`, `digikala.com`, `telewebion.com` (все в белом списке)

## 🌐 Почему иранские Host заголовки работают (Domain Fronting)

Когда конфиг использует `Host: myket.ir`, DPI видит:
```
GET / HTTP/1.1
Host: myket.ir   ← DPI читает это и говорит "ОК, иранский магазин приложений, пропускаем"
```

Реальное TCP соединение идёт на VPN сервер — но DPI не смотрит глубже.

**Лучшие иранские домены для использования в Host:**
- `myket.ir` — иранский Google Play, никогда не будет заблокирован
- `aparat.com` — иранский YouTube, государство его терпит
- `digikala.com` — иранский Amazon
- `telewebion.com` — государственное ТВ, всегда в белом списке
- `anten.ir` — иранские новости

## ⚠️ Честная оценка ситуации

Во время активного военного блэкаута:
- Иранская инфраструктура сама становится нестабильной (таймауты даже на внутренних серверах)
- Любой конфиг со временем умрёт — IP попадают в блэклист
- Решение — **пул конфигов**, а не один идеальный
- Ночью связь лучше (меньше активного глушения)
- Telegram каналы с конфигами обновляющимися каждый час надёжнее любого одного сервера
